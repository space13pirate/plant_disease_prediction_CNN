### :seedling: About this project

<div id="info" align="justify">
  <p style = "text-indent: 50px;">
    The goal of this project is to develop and implement a machine learning model for predicting plant diseases based on images.
  </p>
  <p style = "text-indent: 50px;">
    This repository contains all the necessary files, code, and documentation for replicating the process and deploying the solution.
  </p>

### :file_folder: Project Structure

- **Model Development** – designing and training a Convolutional Neural Network (CNN) model in Google Colab.
- **Web Interface Implementation** – creating a user-friendly web interface using Streamlit.
- **Containerization** – preparing a Docker container for easy deployment and scalability.
